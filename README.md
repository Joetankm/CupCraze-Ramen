# CupCraze Ramen
CupCraze Ramen is a front-end e-commerce website that sells instant cup noodles.  
Built using **HTML**, **CSS**, and **JavaScript**, the site features interactive product listings, a shopping cart, and responsive design.

## Features
- Browse and view various instant noodle products
- Add and remove items from the shopping cart
- Responsive design for desktop and mobile devices
- Interactive UI for smooth user experience

## Tech Stack
- HTML
- CSS
- JavaScript

## Usage
1. Clone the repository:  
   `git clone https://github.com/yourusername/cupcraze-ramen.git`
2. Open `index.html` in your web browser
3. Browse products and interact with the shopping cart
